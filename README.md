
# Login - Register

Basic Authorization Website for user registration and login 

## Features

- Fullscreen mode
- Cross platform

## Run Locally

Go to the project directory

```bash
  cd login-register
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Tech Stack (Client Side)

- HTML
- CSS
  
## Tech Stack (Server Side)

- Node.JS
- Express.JS
- MongoDB
